BIAXIALALL` SYMBOL             

BxPRT

 BxPRT[theta,phi,dieltensVec,einc,esub,IsotropicSub,thickVec,lambda,IsotropicVec] calculates the Jones matrix for Reflection and Transmission from a multi-layer stack of biaxial material at incident and azimutal angles, theta and phi.
The output is of the form:
{reflPRTsp,transPRTsp,kxperk0,kyperk0,kzinc1perk0,kzinc2perk0,kzinc3perk0,kzinc4perk0,kzsub1perk0,kzsub2perk0,kzsub3perk0,kzsub4perk0,a1sub,b1sub,a2sub,b2sub}
where parameters are for light traveling in the negative z direction
The input parameters are of the form:
  theta is the incident angle in radians, phi is the azimuthal angle in radians, lambda is the wavelength (in the same units as the thicknesses)
The multilayer is described by an array of dielectric tensors of the form:
  dieltensVec={dieltens1,dieltens2,dieltens3,...}
  where each element of the array is a 3x3 dielectric tensor or ε=n^2 for isotropic layers. Note that the FIRST tensor in the array is the material CLOSEST TO THE SUBSTRATE

  IsotropicVec is an array with values of 1 or 0 in each element that says whether or not each layer in the stack is isotropic or not.  If the dielectric tensor at a given layer is isotropic (diagonal with all three values equal) then the value of the corresponding element in IsotropicVec should be set to 1, otherwise it is set to 0

The layer thicknesses are described by the array of thickness values of the form:
  thickVec={thick1,thick2,thick3....} or use thickVec={0} for a calculation of a bulk material (in the same units as the wavelength)
  The variable einc is the permitivity (complex index squared) of the incident material (incident material assumed to be isotropic).  
  The variable esub is the permitivity (complex index squared) when the substrate is isotropic and the IsotropicSub is 1 otherwise esub is the dielectric tensor of the anistropic substrate and the IstropicSub is 0.
Note, the coordinate rotation {{1,0},{0,-1}} has been applied to the reflection Jones matrix

The output parameters are of the form:
reflPRTsp = reflection PRT matrix in sp coordinate system
transPRTsp = transmission PRT matrix in sp coordinate system
kxperk0 and kyperk0 are the x and y components of the incident k vector (divided by k0=2π/λ)
kzinc1perk0, kzinc2perk0, kzinc3perk0, and kzinc4perk0 = z component of the k vector in the incident media (for the 4 rays in the incident media)
kzsub1perk0, kzsub2perk0, kzsub3perk0, and kzsub4perk0 = z component of the k vector in the substrate (for the 4 rays in the substrate)
a1sub and b1sub are coefficients describe the electric field of ray 1 in the substrate, E=Ex{1, a1sub, b1sub}
a2sub and b2sub are coefficients describe the electric field of ray 2 in the substrate, E=Ey{a2sub,1 , b2sub}
