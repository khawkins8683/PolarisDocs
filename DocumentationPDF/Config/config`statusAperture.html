CONFIG` SYMBOL             

config`statusAperture

config`statusAperture A user modifiable configuration parameter of Polaris-M. It allows the user to decide whether a ray should continue to propagate if it misses the defined aperture for a surface. Setting this parameter to +3 will cause rays to continue propagation while -3 will cause them to stop when an aperture is missed.

Data Type: Integer (±3)

Default Value: -3

Ray missing an aperture:

If the status is set to 3, then the ray will continue to trace through the rest of the optical system

If a non-sequential ray trace is used, missing the aperture will be reported in the same way as missing the surface since no “next” surface is defined for a non-sequential ray.
