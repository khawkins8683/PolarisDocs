CONFIG` SYMBOL             

config`statusMissed

config`statusMissed A user modifiable configuration parameter of Polaris-M. It allows the user to decide whether a ray should continue to propagate after missing a surface in a sequential ray trace. Setting this parameter to +2 will cause rays to continue propagation while -2 will cause them to stop when a surface has been missed.

Data Type: Integer (±2)

Default Value: -2
