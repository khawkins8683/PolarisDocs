config`parallelizeInput

config`parallelizeInput A user modifiable configuration parameter of Polaris-M. Defined by a Boolean value indicating whether to attempt to utilize multiple kernels on a computer.

Speeds up ray analysis functions like CreateEqualCosinePointSource.

Default Value: False

Note: There is overhead associated with this, so speedy calculations may not benefit
