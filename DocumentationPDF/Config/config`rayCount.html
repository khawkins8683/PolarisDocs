CONFIG` SYMBOL             

config`rayCount

config`rayCount A user modifiable configuration parameter of Polaris-M. It allows the user to decide whether or not a display indicator should be shown for the current ray count, and is useful for large traces.

Data Type: Boolean

Default Value: True

Deprecated Version of Progress Indicator
