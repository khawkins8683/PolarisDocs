DataCentroid

 DataCentroid[imagedatNoDepol_,thresh_] is a lower level centroid algorithm that is used in aligning the simulation parameters to the measured data.  The output for this is a 4 element vector of the form {nx,frac,shiftx,shifty} which can be used as inputs into the function TrueAnglePupil.  This routine is primarily used for setting up the microscope ojective MM images for removal from the measured data, or setting up the Mueller matrices for the glass/air interface at the back of some materials.  The inputs are
imagedat is the Mueller matrix image in the form (Nx,Ny,4,4)
thresh - is the treshold value in the M11 element of the raw data to do the aligning to  (a value of 0.5 is usually a good value for this)
