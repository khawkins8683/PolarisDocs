TrnAOIMuellerSPtest1

 TrnAOIMuellerSPtest1[Tmat_,Pmat_,window_,lambda_,part_]  Generates a MUELLER MATRIX IMAGE corresponding to Transmission from a biaxial MULTILAYER stack in the S-P coordinate basis, defined by the inputs Tmat, Pmat, window, lambda and part.
Tmat,Pmat and window - are arrays that define the range of angles and the exit pupil aperture of the Mueller matrix image (i.e. generated using the TrueAnglePupil2 function)
lambda - the wavelength of light (using the same units as the layer thicknesses in the part definition)
part - Is a 5 element vector of objects of the form {dieltensVec,einc,esub,thickVec,IsotropicVec}.  
Each of these objects is defined as follows:
dieltensVec={dieltens1,dieltens2,dieltens3,...} where each element of the array is a 3x3 dielectric tensor. Note that the FIRST tensor in the array is the material CLOSEST TO THE SUBSTRATE
einc and esub - the permitivity (complex index squared) of the incident and substrate materials.

thickVec={thick1,thick2,thick3....} are the layer thicknesses corresponding to the dielectric tensors in dieltensVec (layer thicknesses and wavelength need to be in the same units)
IsotropicVec - is an array with values of 1 or 0 in each element that says whether or not each layer in the stack is isotropic or not.  If the dielectric tensor at a given layer is isotropic (diagonal with all three values equal) then the value of the corosoponding element in IsotropicVec should be set to 1, otherwise it is set to 0
