BIAXIALALL` SYMBOL             

AlignPolData

 AlignPolData[imagedat_,thresh_,AoIx_,AoIy_:0,objNA_:0.55]  This function alignes the angle range used in the forward simulation (from TrueAnglePupil2) with the Mueller matrix of the exit pupil in the Mueller matrix image data from the polarimeter. The output of this routine is the same as the output of TrueAnglePupil2...That is a vector of arrays of the form {THETAARRAY,PHIARRAY,WINDOW}.  The inputs are as follows:
imagedat is the Mueller matrix image in the form (Nx,Ny,4,4)
thresh - is the treshold value in the M11 element of the raw data to do the aligning to  (a value of 0.5 is usually a good value for this)
AoIx - is the AoI of the microscope objective optical axis (center ray) with respect to the tilt of the part (about the normal to the table)
AoIx - is the AoI of the microscope objective optical axis (center ray) with respect to the tilt of the part (about the vector parallel to the table)
objNA - is the numerical aperture of the microscope objective.
The function AlignPolImages can be run to display 4 images that can be used
to determine the accuracy of the automatic alignment.
