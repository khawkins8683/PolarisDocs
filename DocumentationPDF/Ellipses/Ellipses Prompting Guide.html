ELLIPSES` SYMBOL             

Ellipses`

The Ellipses` package functions can be used to create intricate polarization graphics.  The functions can be used to make polarization graphic primitives for the pre-existing Mathematica graphics package.  These primitives can be used with an extensive array of both 2-D and 3-D Mathematica primitives to create unique diagrams that communicate with the audience interactively.

Package Functions

Ellipse — This function builds a 2-D polarization ellipse graphics primitive from a Jones vector

Ellipse3D — This function builds a 3-D polarization ellipse graphics primitive from a Polarization vector

PoincareEllipses — This function builds multiple 3-D polarization ellipse graphics primitives located on a Poincare Sphere.

PolarizationVector — This function returns the Polarization vector and PoincareSphere coordinates for a given Stokes vector.
