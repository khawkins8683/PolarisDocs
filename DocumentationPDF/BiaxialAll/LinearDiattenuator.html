LinearDiattenuator

 LinearDiattenuator[d_,φ_,win_] Mueller matrix image of a diattenuator.  Inputs are 3 arrays

d - array where each point is a diattenuation magnitude

φ - array where each point is a diattenuation orientation

win - is a binary array defining the aperture of the exit pupil
