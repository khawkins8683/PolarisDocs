CONFIG` SYMBOL             

config`statusAbsorb

config`statusAbsorb A user modifiable configuration parameter of Polaris-M. It allows the user to decide whether a ray should continue to propagate after intersecting a surface with sys`mode == “Absorb”. Setting this parameter to +8 will cause rays to continue propagation while -8 will cause them to stop at absorptive surfaces.

Data Type: Integer (±8)

Default Value: -8
