CONFIG` SYMBOL             

config`statusUnknown

config`statusUnknown A user modifiable configuration parameter of Polaris-M. It is used when Polaris is unable to determine the state of a ray. It can be changed, but doing so is strongly recommended against.

Data Type: Integer (0)

Default Value: 0

This status is the indication that there was an issue with the ray trace.  No know examples exist where a status of 0 is created.  Please notify Airy Optics if a ray`status of 0 appears in your ray data.
