config`statusInhibitedReflect

config`statusInhibitedReflect A user modifiable configuration parameter of Polaris-M. It allows the user to decide whether a ray should continue to propagate after experiencing inhibited reflection in a crystalline material. Setting this parameter to +6 will cause rays to continue propagation while -6 will cause them to stop on inhibited reflection.

Data Type: Integer (±6)

Default Value: -6

This status is thrown in the same way as , but occurs inside of anisotropic materials.
