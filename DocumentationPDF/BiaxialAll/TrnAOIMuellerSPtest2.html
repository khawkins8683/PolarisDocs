TrnAOIMuellerSPtest2

 TrnAOIMuellerSPtest2[Tmat_,Pmat_,window_,lambda_,part_]  Generates a MUELLER MATRIX IMAGE corresponding to Transmission from a biaxial SINGLE -LAYER  in the S-P coordinate basis, defined by the inputs Tmat, Pmat, window, lambda and part.
Tmat,Pmat and window - are arrays that define the range of angles and the exit pupil aperture of the Mueller matrix image (i.e. generated using the TrueAnglePupil2 function)
lambda - the wavelength of light (using the same units as the layer thicknesses in the part definition)
part - Is a 5 element vector of objects of the form {dieltensVec,einc,esub,thickVec,IsotropicVec}.  Each of these objects is defined as follows:

dieltensVec={dieltens}
where dieltens is a 3x3 dielectric tensor.

einc and esub - the permitivity (complex index squared) of the incident and substrate materials.

thickVec={thick1} is the layer thicknesses (layer thicknesses and wavelength need to be in the same units)


IsotropicVec = {Isotropic} where Isotropic has a value of 1 or 0 that says whether or not the layer is isotropic or not.  If the dielectric tensor is isotropic (diagonal with all three values equal) then the value of Isotropic should be set to 1, otherwise it is set to 0
