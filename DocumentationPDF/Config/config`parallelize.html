CONFIG` SYMBOL             

config`parallelize

config`parallelize A user modifiable configuration parameter of Polaris-M. If set to True, it allows the user to attempt to utilize multiple kernels on a computer. There is overhead associated with this, so speedy calculations may not benefit.

Data Type: Boolean

Default Value: False
