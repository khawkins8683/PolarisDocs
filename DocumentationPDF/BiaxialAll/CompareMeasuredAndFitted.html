CompareMeasuredAndFitted

 CompareMeasuredAndFitted[muellersim_,muellerdata_,rngvar_:All,pgtf_:True,colorval_:Hue[0]]  Draws a 4x4 array of plots that compares the measured and fitted mueller matrix data.  Inputs are as follows:
muellersim - an array of dimensions {N,4,4} that has the simulated mueller matrices
muellerdata - an array of dimensions {N,4,4} that has the measured mueller matrices
rngvar - the range for all of the plots (rngvar=All autoscales all of the plots)
pgtf - a True/False handle for displaying the measured data (pgtf=True connects the data together, pgtf=False shows the measured data as points)
colorval - Color the measured data will be.
