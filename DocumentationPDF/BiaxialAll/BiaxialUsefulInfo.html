Notes on BiaxialMMsim part of the package:
This software is intended for use in simulating angle of incidence Mueller matrix images taken from the polarization lab' s MM polarimeter system.  It contains all the necessary functions required to simulate a Mueller matrix image from reflection or transmission for an anisotropic material.


Notes on BiaxialV11 part of the package:
This software package is intended to calculate the transmission and reflection properties from single layer and multilayer thin film stacks comprised of isotropic and anisotropic material.  Each layer in the thin film stack is described using a 3x3 dielectric tensor.
All data is expressed in the s - p coordinate system

The three most general routines in the package are :
BxMlJ   Calculates both the reflection and transmission Jones matrix of an anisotropic multilayer stack.
BxMlRefJ   Calculates the Jones matrix from reflection of an anisotropic multilayer stack.
BxMlTrnJ   Calculates the Jones matrix from transmission of an anisotropic multilayer stack.

The results of these calculations can be converted into Mueller matrices using the function :
BxJ2M0   Converts Jones matrices to Mueller matrices using the correct sign convention for our polarimeter.
Additional Routines are built in that are optimized for Single layer calculations.  Discriptions are in the usage statements.

A couple of things to note.
Software doesn't work with optically active dielectric tensors (Yet). Although it can be modified to do so.
Trying to calculate the Jones or Mueller matrix from a structure when theta = 0 and/or phi = 0 will result in Junk output.  This is because the x and y E field coupling terms in the algorithm end up being 0/0. A simple workaround is to use small values that are close, but not, equal to zero.You need to be careful though because if you use numbers that are to close to zero (10^-10) there are some instabilities. It turns out that the instability is alot worse with theta close to zero than with phi close to zero. I've found that values of around 10^-6 are usually satisfactory.  Also when you calculate pseudo - normal incidence (theta = 10^-6), it is best to have a non - zero value for the azimuthal angle phi.
