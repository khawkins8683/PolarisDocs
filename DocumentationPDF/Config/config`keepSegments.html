CONFIG` SYMBOL             

config`keepSegments

config`keepSegments A user modifiable configuration parameter of Polaris-M. It allows the user to select a ray subset and discard all other rays. Its main use is to reduce memory consumption.

Data Type: String (Conditional statement in quotes)

Default Value: “ray`ID[[1]]>0”
