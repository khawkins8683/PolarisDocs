IsoRCWA Raytrace

Set up the Optical System.Diffractive optical elements are defined in the coating inputs for an optical system. To use this coating, it is required that sys`coating`type → “RCWA”.  Coating type “RCWA” designates a Rigorous Coupled Wave Analysis (diffraction grating) calculation is associated with a surface.

The rigorous coupled wave analysis [RCWA] routine describes the reflection and refraction at a ray intercept for periodic structures including diffraction gratings and some holograms.
For internal reflection, the incident material has a higher refractive index than the exiting material, and beyond the critical angle, total reflection occurs. For external reflection, the incident material has a lower refractive index.
For the case of gratings, the grating equation; must be satisfied.

    • sys`coating`name → ""

Default:  Empty string
Type:  string
Description:  This input can be used to label the coating.  
Example:  “Diffraction Grating”

    • sys`coating`RCWA`ordersCalculate → 20

Default: 20
Type: non-negative integer
Description:    Specify the number of orders to calculate durring the ray trace.
Notes:  Larger integers will increase the computation time

    • sys`coating`RCWA`ordersReturned → {-2,-1,0,1,2}

Default:  {-2,-1,0,1,2}
Type:  1-D array of integers
Description: list of the diffraction orders returned in the ray data
Notes: Do not have more modes than specified in ordersCalculate

    • sys`coating`RCWA`gratingPeriod → 0.5

Default:  0.5
Type: real number in μm
Description:  spatial distance between repeating elements of grating

    • sys`coating`RCWA`thicknessVec → {0.125, 0.125}

Default:  {0.125, 0.125}  
Type:  1-D array of real numbers in μm
Description:  list of thicknesses for each layer; the last element correspond to layer next to substrate

    • sys`coating`RCWA`widthMatrix → {{0,0.4,0.6,1},{0,0.4,0.6,1}}

Default:  {{0, 0.4, 0.6, 1}, {0, 0.4, 0.6, 1}}
Type: 2-D array of real numbers in mm
Description:  list describing start and stop positions for each block, normalized between 0 and 1; the last element correspond to layer next to substrate.

    • sys`coating`RCWA`indexMatrix → {{1, 1.5, 1}, {1, 1.5, 1}}

Default:  {{1, 1.5, 1}, {1, 1.5, 1}}
Type:  2-D array of complex numbers (unitless)
Description:  list of refractive indices for each block specified in widthMatrix; the last element correspond to layer next to substrate.

    • sys`coating`RCWA`xLocal → {1,0,0}

Default:  {1,0,0}
Type: 3 element array in mm
Description:  3D vector specifying the direction (in global coordinates) that contains the grating repetitions.
