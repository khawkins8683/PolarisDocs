ELLIPSES` SYMBOL             

Ellipse3D

  A function that returns graphics primitives forming an ellipse corresponding to the 3-dimensional Jones vector, {ex, ey, ez}, default is the origin.

Inputs:
     {ex,ey,ez} - the 3D Jones vector
     {x,y,z} - the location of the center of the ellipse in global coordinates
     opts - optional parameters to change the ellipse appearance

Output:
     a list of values for the ellipse position as well as any visual changes to the ellipse

The apparent colors of the ellipse are affected by the lighting - set Lighting->Neutral in the Show statement to get the defined colors.

Use the Arrowheads graphics primitive to change the appearance of the arrowheads.   

Opts`θStart and Opts`StartPhase, and Opts`θEnd and Opts`EndPhase are interchangeable.

Options  (9)

Opts`Color

Default Value: "LinearRainbow"
an option that allows the user to change the color of the ellipse

PlotPoints

Default Value: 50
an option that defines the number of points plotted

Opts`θStart

Default Value:  0
an option that allows sets the start phase (in radians) for the ellipse.

Opts`θEnd

Default Value:  2π
an option that allows sets the end phase (in radians) for the ellipse.

Opts`RadiusFunction

Default Value:   (Thickness[0.005]&)]
an option that changes the thickness of the ellipse.

Opts`OpacityFunction

Default Value:  Opaque
an option that will change the opacity of the ellipse.

Opts`ThickFunction

Default Value:  (Thickness[0.005]&)
an option that changes the thickness of the ellipse.

Opts`PreformanceGoal

Default Value:  Speed
an option that  will change target performance attributes.

Opts`Phase

Default Value:  "Decreasing"
an option that  will change phase convention of the ellipse.
