CONFIG` SYMBOL             

config`stopRay

config`stopRay A config parameter used in conjunction with the stopRay Function. If config`stopRayFunction evaluates to True for a given ray segment, the ray will be given a status of -7 and be terminated. Otherwise it will be assigned a value of +7 and continue to propagate.

Data Type: Integer of value ±7

Default Value: -7
