CONFIG` SYMBOL             

config`statusTIR

config`statusTIR A user modifiable configuration parameter of Polaris-M. It allows the user to decide whether a ray should continue to propagte after experiencing total internal reflection. Setting this parameter to +4 will cause rays to continue propagation while - 3 will cause them to stop when TIR is experienced.

Data Type: Integer (±4)

Default Value: -4
